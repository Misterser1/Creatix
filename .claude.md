# Правила проекта Creatix

## Общие правила
- **Язык общения**: Всегда пишу на русском
- **При потере контекста**: Читаю этот файл и смотрю скриншот-референс

## Проект
- **Название компании**: Creatix
- **Деятельность**: Разработка сайтов, CRM и ERP систем
- **Стиль**: Современный, тёмный, с 3D эффектами как на референсе

## Референс
- **Скриншот**: `reference/opticore-design.png`
- Навигация как на референсе
- Те же цвета (фиолетовый/розовый/голубой градиенты)
- 3D "жидкие" формы
- Тёмный фон

## Проверка результата
После создания/изменения страницы:
1. Запустить сервер ScreenCreate: `node ScreenCreate/server.js`
2. Сделать скриншот через API: `http://localhost:3500/screenshot?url=file:///путь/к/файлу`
3. Сравнить с референсом
4. Доработать пока не будет 100% соответствие
